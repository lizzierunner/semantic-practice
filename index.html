Basic Text Tags

  • p - for paragraph text, usually multiple lines
  • em - for text that needs to be emphasized
  • strong - for important text
  • cite - for the title of a work
  • mark - highlighted text
  • b - text that needs attention
  • i - text in an alternative voice
  • u - underlined text
  • kbd - keyboard input
  • samp - sample computer output
  • var - variables in a program

When choosing a tag pair to wrap your content in, be sure to think about what your text actually is.

Remember: HTML tags aren't used to style the text a certain way. Use tags like strong or em only when they add meaning.

Tags for Grouping and Describing Content

  • h1 - most important heading, only one per page
  • h2 - second level of heading
  • h3 - third level of heading
  • h4 - fourth level of heading
  • h5 - fifth level of heading
  • h6 - sixth level of heading
  • section - container for grouping related content

Headings are used like an outline. Each section should have one heading that matches the topic.

The section tag groups related content and can be nested inside other sections with headings.

Special Characters

  • © - copyright symbol
  •   - non-breaking space
  • & - ampersand
  • < and > - less than and greater than symbols

Use entity characters to prevent display issues. For example, use & instead of & to avoid breaking HTML.

Lists

  • ol - ordered lists for steps or sequences
  • ul - unordered lists for grouped but unordered content
  • li - individual list items
